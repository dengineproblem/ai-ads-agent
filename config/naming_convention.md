# Naming Convention для объявлений

Система именования позволяет группировать объявления по креативам (видео/изображениям).

---

## Зачем это нужно

Одно видео может использоваться в нескольких объявлениях:
- Разные аудитории (возраст, гео, lookalike)
- Разные adsets
- Разные кампании

**Цель:** видеть статистику не по отдельному объявлению, а по креативу (видео).

---

## Формат имени Ad

```
{creative_tag}_{описание}
```

| Часть | Описание |
|-------|----------|
| `creative_tag` | Идентификатор видео/изображения (первая часть до `_`) |
| `описание` | Любая информация: аудитория, гео, заметки |

---

## Примеры

```
Видео: kitchen.mp4 → тег "kitchen"

Объявления:
- kitchen_30-45_msk
- kitchen_lookalike
- kitchen_retarget_7d

Все объявления группируются по тегу "kitchen"
```

```
Видео: promo_january.mp4 → тег "promo"

Объявления:
- promo_cold_audience
- promo_warm_audience
- promo_lookalike_3pct
```

---

## Как парсить

```python
def get_creative_tag(ad_name):
    """Первая часть имени до _ = тег креатива"""
    return ad_name.split("_")[0]
```

```
Ad Name: "kitchen_lookalike_3pct"
Creative Tag: "kitchen"
```

---

## Группировка

Все объявления с одинаковым `creative_tag` группируются:

```
Креатив: kitchen
├── kitchen_30-45_msk      $100, 20 leads
├── kitchen_lookalike      $150, 35 leads
└── kitchen_retarget       $80, 15 leads
────────────────────────────────────────
ИТОГО:                     $330, 70 leads, CPL $4.71
```

---

## Правила

### Обязательно
- `creative_tag` = первое слово в имени
- Разделитель: `_`
- Латиница (без кириллицы)

### Рекомендуется
- Короткий тег (1 слово, до 15 символов)
- Тег = название видео файла без расширения
- Описание после тега — любое понятное тебе

## Использование в Skills

### /creative-analyzer
- Группирует ads по `creative_tag`
- Показывает агрегированные метрики по каждому видео
- Risk Score по креативу

### /ads-reporter
- Breakdown по креативам в отчётах

### /campaign-manager
- При создании ad использует тег из `config/creatives.md`
