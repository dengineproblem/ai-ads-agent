# Справочник Geo-локаций для Facebook Ads

## Как использовать

1. При онбординге — ищи города в таблицах ниже
2. Если города нет — вызови MCP: `search_geo_locations(query="город", location_types=["city"])`
3. Добавь найденные ID в бриф (секция "Гео-локации")

## JSON формат для API

### Только страны:
```json
{
  "countries": ["KZ", "RU"]
}
```

### Города с радиусом:
```json
{
  "cities": [
    {"key": "1289662", "radius": 25, "distance_unit": "kilometer"},
    {"key": "1301648", "radius": 25, "distance_unit": "kilometer"}
  ]
}
```

### Города + исключение стран:
```json
{
  "cities": [{"key": "1289662", "radius": 40}],
  "excluded_countries": ["BY"]
}
```

---

## Как найти City Key

1. Вызови MCP tool:
   ```
   search_geo_locations(query="Название города", location_types=["city"])
   ```

2. В ответе найди:
   - `key` — это City Key
   - `region_id` — ID региона (если есть)
   - `country_code` — код страны

3. Добавь в этот справочник для будущего использования
