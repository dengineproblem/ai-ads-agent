# Facebook Ads Best Practices

Лучшие практики для эффективной рекламы в Facebook/Instagram.

---

## Структура аккаунта

### Рекомендуемая иерархия

```
Account
└── Campaign (по цели/направлению)
    └── AdSet (по аудитории/таргетингу)
        └── Ad (по креативу)
```

### Правила именования

```
Campaign: {Направление}_{Цель}_{Дата}
AdSet: {Аудитория}_{Возраст}_{Гео}
Ad: {Креатив}_{Версия}
```

**Примеры:**
- Campaign: `Кухня_Leads_2025-01`
- AdSet: `Интересы_25-45_KZ`
- Ad: `Carousel_v1`

## ODAX Objectives

Facebook использует Outcome-Driven Ad Experiences (ODAX).

| Objective | Когда использовать | Оптимизация |
|-----------|-------------------|-------------|
| **OUTCOME_AWARENESS** | Охват, узнаваемость бренда | Reach, Impressions |
| **OUTCOME_TRAFFIC** | Трафик на сайт | Link Clicks, Landing Page Views |
| **OUTCOME_ENGAGEMENT** | Вовлечение, лайки, комменты | Post Engagement |
| **OUTCOME_LEADS** | Лид-формы, WhatsApp | Leads, Conversations |
| **OUTCOME_SALES** | Покупки на сайте | Purchases, Add to Cart |
| **OUTCOME_APP_PROMOTION** | Установки приложения | App Installs |

## Таргетинг

### Размер аудитории

| Размер | Рекомендация |
|--------|--------------|
| < 100K | Слишком узко, расширить |
| 100K - 500K | Хорошо для теста |
| 500K - 2M | Оптимально |
| 2M - 10M | Хорошо для масштабирования |
| > 10M | Может быть слишком широко |

### Advantage+ Audience

- Facebook автоматически находит аудиторию
- Рекомендуется для большинства случаев
- Настраивается через `targeting_automation.advantage_audience=1`

### Lookalike Audiences

| Ratio | Размер | Использование |
|-------|--------|---------------|
| 1% | Самые похожие | Лучшая конверсия, малый охват |
| 1-3% | Похожие | Баланс качества и охвата |
| 3-5% | Средне похожие | Масштабирование |
| 5-10% | Широкие | Максимальный охват |

## Бюджеты

### Campaign Budget Optimization (CBO)

**Когда использовать:**
- 3+ adsets в кампании
- Adsets с похожей эффективностью
- Нужно автоматическое распределение

**Когда НЕ использовать:**
- Тестирование новых аудиторий
- Разные цели у adsets
- Нужен контроль над бюджетом каждого adset

### Минимальные бюджеты

| Оптимизация | Минимум |
|-------------|---------|
| Impressions/Reach | $1/день |
| Link Clicks | $2/день |
| Conversions | $5/день (рекомендуется $10+) |
| App Installs | $5/день |

### Learning Phase

- Требуется ~50 конверсий за 7 дней
- Минимальный бюджет для выхода: `50 * target_CPA / 7` в день
- Не менять бюджет более чем на 20% во время learning

## Креативы

### Форматы

| Формат | CTR | Когда использовать |
|--------|-----|-------------------|
| **Single Image** | Средний | Простые офферы, тесты |
| **Carousel** | Высокий | Несколько продуктов, storytelling |
| **Video** | Высокий | Сложный продукт, эмоции |
| **Collection** | Высокий | E-commerce каталог |

### Рекомендации по размерам

| Placement | Ratio | Размер |
|-----------|-------|--------|
| Feed | 1:1 | 1080x1080 |
| Stories | 9:16 | 1080x1920 |
| Reels | 9:16 | 1080x1920 |

### Текст

- **Primary text**: 125 символов (первые 2 строки)
- **Headline**: 27 символов
- **Description**: 27 символов
- Используй эмодзи умеренно
- Один CTA на креатив

## A/B тестирование

### Что тестировать

1. **Креативы** - разные изображения/видео
2. **Тексты** - разные заголовки и описания
3. **Аудитории** - разные таргетинги
4. **Placements** - Feed vs Stories vs Reels

### Правила теста

- Тестировать ОДНУ переменную за раз
- Минимум 1000 impressions на вариант
- Минимум 7 дней теста
- Статистическая значимость > 95%

### Определение победителя

```
Winner: лучший CPL/CPA при достаточном объеме данных
Looser: остановить если CPL > 2x от победителя
```

## Scaling (Масштабирование)

### Вертикальное (бюджет)

1. Увеличивать бюджет на **20-30%** каждые 2-3 дня
2. Не увеличивать во время learning phase
3. Мониторить CPL после каждого увеличения

### Горизонтальное (аудитории)

1. Дублировать успешный adset
2. Менять только аудиторию
3. Тестировать на малом бюджете ($10-20)
4. Масштабировать победителей

## Типичные проблемы

### Высокий CPM

**Причины:**
- Узкая аудитория
- Высокая конкуренция
- Плохое качество креатива

**Решения:**
- Расширить аудиторию
- Тестировать другие интересы
- Обновить креативы

### Низкий CTR

**Причины:**
- Неинтересный креатив
- Нерелевантная аудитория
- Плохой текст

**Решения:**
- Новые креативы
- A/B тест текстов
- Уточнить таргетинг

### Высокий CPL/CPA

**Причины:**
- Плохая конверсия лендинга
- Нерелевантный трафик
- Слабый оффер

**Решения:**
- Оптимизация лендинга
- Уточнить таргетинг
- Усилить оффер

## Сезонность

### Дорогие периоды

| Период | CPM рост |
|--------|----------|
| Black Friday | +50-100% |
| Новый год | +30-50% |
| 8 марта | +20-30% |
| Конец квартала | +10-20% |

### Рекомендации

- Увеличить бюджет заранее
- Подготовить креативы за 2 недели
- Ожидать выше CPM, планировать бюджет
