# Реестр креативов

Маппинг creative_tag к описанию и файлам.

---

## Формат записи

```markdown
### {creative_tag}
- **Описание**: Что показывает креатив
- **Тип**: video / image / carousel
- **Файл**: путь к исходному файлу (опционально)
- **Дата создания**: YYYY-MM-DD
- **Статус**: active / paused / archived
- **Заметки**: особенности
```
